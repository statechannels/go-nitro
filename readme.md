Implementation of nitro protocol in golang.

---

### For developers
To format:
```shell
gofmt -w .
```
To lint:
```shell
golangci-lint run
```
To build:
```shell
go build ./...
```
To run tests:
```shell
go test ./...
```
To view docs website:
```shell
godoc --http :6060
```
and navigate to http://localhost:6060/pkg/github.com/statechannels/go-nitro/
